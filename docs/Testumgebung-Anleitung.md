# Testumgebung Setup - Schritt für Schritt Anleitung

## Übersicht

Diese Anleitung führt dich durch die Einrichtung einer vollständigen Testumgebung mit CI/CD-Pipeline und automatischen Prüfungen für das Projekt.

---

## 1. Vorbereitung

### 1.1 Repository-Einstellungen überprüfen

- Gehe zu **Settings** > **General**
- Stelle sicher, dass das Repository öffentlich oder privat nach Bedarf konfiguriert ist
- Notiere die Repository-URL für späteren Gebrauch

### 1.2 Erforderliche Zugriffsrechte

- Stelle sicher, dass du Admin-Rechte im Repository hast
- Alle Team-Mitglieder sollten mindestens Write-Zugriff haben
